Put homepage gallery images in this folder.

Recommended workflow:
1. Add or replace image files here.
2. Edit src/data/homepage-gallery.json.
3. Commit and push to GitHub.
4. Wait for GitHub Pages to redeploy.

Example JSON item:
{
  "image": "images/homepage/your-image.jpg",
  "alt": "Image description",
  "title": "Optional title",
  "description": "Optional caption"
}
